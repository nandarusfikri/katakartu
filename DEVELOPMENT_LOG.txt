## 🧪 How to Test

1. Run server:
   ```bash
   cd Game_KataBaku
   go run cmd/server/main.go
   ```

2. Open browser: http://localhost:8080

3. Create room (Player 1):
   - Click "Buat Room"
   - Enter username
   - Click "Buat Room"

4. Join room (Player 2):
   - Open new tab
   - Click "Join Room"
   - Enter same room code
   - Click "Join"

5. Start game (Host):
   - Click "Mulai Game"

6. Play:
   - Main Card + kartu dari tangan → Bentuk kata
